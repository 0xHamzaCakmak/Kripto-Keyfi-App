# Kripto Keyfi

Kripto portfoy takibi, ekosistem kesfi ve topluluk ekranlarini tek bir arayuzde toplayan Vite + React uygulamasi.

## Gelistirme

Gereksinim: Node.js 20+

1. Bagimliliklari kurun: `npm install`
2. Ortam degiskenlerini hazirlayin: `.env.example` dosyasini `.env.local` olarak kopyalayin
3. Gerekliyse `GEMINI_API_KEY` degerini `.env.local` icinde guncelleyin
4. Gelistirme sunucusunu baslatin: `npm run dev`

Uretim paketi icin:

- `npm run build`
- `npm run preview`
